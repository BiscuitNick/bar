# Retro Games Hub

A collection of classic retro games built with Next.js 15, React 19, and Tailwind CSS.

## Games

- **Tetris**: The classic block-stacking puzzle game.
- **Snake**: Eat the food, grow your snake, and avoid the walls!
- **Minesweeper**: Clear the board without detonating any mines.

## Getting Started

First, install the dependencies:

```bash
pnpm install
```

Then, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Running Tests

This project uses Vitest for testing. To run the test suite:

```bash
pnpm test
```

To run tests in watch mode:

```bash
pnpm test:watch
```
